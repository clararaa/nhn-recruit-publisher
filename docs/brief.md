# Project Brief: NHN 채용공고 퍼블리셔

## Executive Summary

NHN 채용팀이 자사 채용사이트(careers.nhn.com) 공고를 외부 10개 채용 플랫폼에 동일한 서식으로 빠르게 게재할 수 있도록, 공고 텍스트를 입력하면 인라인 스타일 HTML과 이미지를 자동 생성해주는 웹 도구.

## Problem Statement

- 채용 공고를 최대 10개 외부 플랫폼에 게재할 때, 사이트마다 에디터가 달라 복사/붙여넣기 시 **서식이 깨지는 문제** 발생
- 공고 1건당 전체 플랫폼 게재에 약 **1시간** 소요 (소수 사이트만 올릴 경우 ~10분)
- 월 20~30건 공고 × 최대 1시간 = **월 20~30시간**의 반복 작업
- 7명의 채용담당자가 각자 동일한 비효율을 겪고 있음

## Proposed Solution

자사 채용사이트 공고 내용을 붙여넣으면:

1. **인라인 스타일 HTML** 생성 — 어떤 에디터에 붙여넣어도 서식이 깨지지 않음
2. **이미지(PNG/JPG)** 생성 — 이미지 업로드를 지원하는 플랫폼용
3. NHN Careers 디자인 톤(블랙 & 화이트 미니멀) 유지
4. 너비 ~700px 기준의 세로형 공고문 레이아웃

## Target Users

### Primary: NHN 채용팀 (7명)
- 채용담당자로서 공고 작성 및 외부 플랫폼 게재 담당
- 반복적인 서식 맞춤 작업에 많은 시간 소요
- 기술적 전문성보다는 빠르고 직관적인 도구 필요

## Goals & Success Metrics

### Business Objectives
- 공고 게재 시간을 1시간 → 10분 이내로 단축 (80% 이상 절감)
- 10개 플랫폼에서 동일한 서식 품질 보장
- 채용팀 전원(7명)이 즉시 사용 가능한 직관적 UX

### Key Performance Indicators
- 공고 1건 전체 플랫폼 게재 소요 시간
- HTML 서식 깨짐 발생 빈도 (목표: 0건)
- 팀원 도구 채택률

## MVP Scope

### Core Features (Must Have)
- **텍스트 입력 영역**: 자사 채용사이트 공고 내용 붙여넣기
- **섹션 자동 인식**: 표준 공고 섹션(회사소개, 팀소개, 주요업무, 자격요건 등) 자동 파싱
- **HTML 생성**: 인라인 스타일 적용된 HTML 코드 출력 + 원클릭 복사
- **이미지 생성**: 동일 디자인의 PNG/JPG 이미지 다운로드
- **미리보기**: 생성된 공고문 실시간 미리보기
- **NHN 스타일 디자인**: 블랙 & 화이트 톤, 볼드 타이포, 미니멀 디자인 (상단 법인별 배너 이미지는 제외 — NHN, NHN Cloud, NHN Dooray 등 관리법인이 다양하므로 텍스트 콘텐츠만 변환)

### Out of Scope for MVP
- 각 플랫폼 API 연동 (자동 게재)
- 공고 이력 저장/관리
- 사용자 계정/인증 시스템 (사내망 접근 제한으로 대체)
- 다국어 지원
- 모바일 최적화

## Post-MVP Vision

### Phase 2
- 공고 템플릿 저장/불러오기
- 공고 이력 관리 (어떤 플랫폼에 게재했는지 추적)
- 플랫폼별 최적화 옵션 (각 사이트 에디터 특성 반영)

### Long-term Vision
- 채용 플랫폼 API 연동으로 원클릭 동시 게재
- 공고 성과 추적 (조회수, 지원수)

## Technical Considerations

### Platform Requirements
- **배포**: Vercel
- **접근 제한**: 미정 (사내망 제한 또는 공개 접근 — 추후 결정)
- **브라우저**: Chrome 최신 버전 (사내 표준)

### Architecture Considerations
- 프론트엔드 중심 SPA (백엔드 최소화)
- HTML-to-Image 변환은 클라이언트 사이드 처리 가능 (html2canvas 등)
- 인라인 스타일 HTML은 클라이언트에서 생성

## Constraints & Assumptions

### Constraints
- 접근 제한 방식 미정 (사내망 제한 또는 공개)
- Vercel 무료/Pro 플랜 활용
- 외부 서비스 의존도 최소화

### Key Assumptions
- 공고 섹션 구조가 대부분 동일함
- 외부 플랫폼 에디터가 인라인 스타일 HTML을 허용함
- 이미지 업로드 시 용량/크기 제한은 플랫폼별로 상이하나, 일반적으로 5MB 이내
